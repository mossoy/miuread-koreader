# MiuRead v0.1.0-dev.14

## 本版定位

这是公开仓库的首个 OTA 引导版本。该版本需要手动安装一次；从后续版本开始，可通过插件内“检查更新”进行升级。

## OTA 更新

- 固定使用单一公开更新清单：
  `https://raw.githubusercontent.com/miumiupy98-art/miuread-koreader/main/update.json`
- 删除公开版中的正式/内测通道切换。
- 删除手动填写更新清单地址的入口。
- 更新清单兼容 0.3.6.7 使用的字段：
  - `version`
  - `name`
  - `notes`
  - `package_url`
  - `sha256`
  - `delete_list`
- 下载 OTA 包后强制校验 SHA-256。
- 校验 OTA ZIP 根目录必须为 `miuread.koplugin/`。
- 安装前备份当前插件。
- 安装失败时恢复旧插件。
- 支持安全的 `delete_list`，拒绝绝对路径、反斜杠和 `..` 路径。
- 安装成功后提示完全退出并重新启动 KOReader。

## 数据保留

用户数据不保存在插件安装目录内，因此 OTA 不会覆盖：

- 微信读书登录状态与 Cookie；
- 已下载书籍；
- 封面缓存；
- 本地阅读位置；
- 阅读同步状态；
- 插件设置。

## 其他

- 保留 v0.1.0-dev.13 的 UI、评论窗口、阅读时长和阅读进度同步逻辑。
- 版本号升级为 `0.1.0-dev.14`。
- 设置中的“更新与关于”仅保留“检查更新”、当前版本和关于。
