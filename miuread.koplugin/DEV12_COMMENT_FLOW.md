# MiuRead 0.1.0-dev.12 · 评论连续分页与空间利用优化

本版本只调整评论窗口，不修改阅读时间同步、搜索、书架、封面、下载和更新通道。

- 评论由预分组页面改为单一连续 HTML 文档，由 KOReader 根据实际内容高度自然分页。
- 删除左上角 `1/5` 等插件预分页数字。
- 实体前后翻页键直接滚动连续评论文档。
- 点击评论卡片内部进入下一页。
- 点击评论卡片外部关闭弹窗，并拦截事件，避免底层正文被点击。
- 右上角 `×` 改为覆盖式小按钮，不再占据整行标题栏。
- 评论卡片增大，减少边距和内边距，提高有效显示面积。
- 打开、翻页和关闭仍限定在评论卡片区域局部刷新。
